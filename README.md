# 🚀 Anish Das | Personal Portfolio

A modern, dynamic, and interactive personal portfolio website showcasing my skills, projects, and coding journey as a 3rd-year CSE (AI&ML) student at GITAM University. Built with raw HTML, CSS, and JavaScript without heavy frameworks, focusing on performance, glassmorphism aesthetics, and real-time data integrations.

## ✨ Features

- 🎨 **Modern Glassmorphism UI:** Beautiful dark theme with vibrant gradients, blurred cards, and custom CSS animations.
- 🎵 **Live "Now Playing" integration:** Displays my real-time Spotify listening status using the Last.fm API.
- 📊 **Real-time LeetCode Stats:** Fetches my actual solved problems dynamically from LeetCode.
- 💻 **Interactive Code Snippets:** Floating, animated code cards on the hero section.
- 📱 **Fully Responsive:** Adapts seamlessly to all screen sizes (mobile, tablet, desktop).
- ⚡ **Lightweight & Fast:** Pure HTML/CSS/JS means zero bundling wait times and instant loads.

## 🛠️ Technologies Used

- **HTML5:** Semantic structuring
- **CSS3:** Custom properties (variables), Flexbox/Grid, Animations, Backdrop-filters
- **JavaScript (ES6+):** DOM manipulation, Async/Await API fetching
- **APIs:**
  - [Last.fm API](https://www.last.fm/api) for live Spotify track tracking
  - [Alfa LeetCode API](https://alfa-leetcode-api.onrender.com/) for dynamic competitive programming stats
